Makes the HTML element [\<dialog\>] draggable.

## Quick start

Install

```shell
pnpm add dialog-draggable
# yarn add dialog-draggable
# npm i dialog-draggable
```

make `<dialog>` draggable

```typescript
import { makeDialogDraggable } from 'dialog-draggable';

makeDialogDraggable();
```

## Develop

```shell
pnpm install
```

### local dev preview

```shell
pnpm dev
```

### test

```shell
pnpm test
```
